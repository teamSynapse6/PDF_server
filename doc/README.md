# DocEx
각종 문서에서 텍스트를 추출하는 모듈입니다.
현재 지원하는 문서는 다음과 같습니다.

- PDF
- HWP
- PPTX
- DOCX

# Dependency
```
$ pip install -r requirements.txt
```

# Get Started
```
$ python pdf.py filename
$ python hwp.py filename
$ python docx.py filename
$ python pptx.py filename
```
